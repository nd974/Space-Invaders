# CLUED'IUTO

CLUED'IUTO est une variante du jeu Cluedo basée sur l'IUT Informatique d'Orléans


### Prérequis

Avoir Python3
```sh
$ sudo apt-get update
$ sudo apt-get install python3.6
```

### Lancement

```sh
$ python3 cluedoTexte.py
ou
$ ./run.sh
```
