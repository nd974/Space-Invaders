# Space Invaders

Space Invaders est un jeu Java avec du graphisme ASCII

### Prérequis

Avoir Java
```sh
$ sudo apt-get update
$ sudo apt-get install openjdk-8-jre
```

### Lancement

```sh
$javac *.java
$java Executable
ou
$ ./run.sh
```

### Commandes
- Déplacement: fleches du clavier
- Tirer: [SPACE] 
- Start: [ENTER]
